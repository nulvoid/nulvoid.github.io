Late Model EXE

This late model EXE is specifically built for racing late models in NR2003. PTA cars have a rev limiter set to 8000, GNS cars have a rev limiter set to 7000, and CTS cars have a rev limiter set to 9000. I personally haven't testing any other cars, but weights are realistic for late model (GNS) and super late model (PTA) racing.
